# Accurat System — Комплексная система управления автомойкой

[![.NET Build](https://github.com/iamfifya/AccuratSystem/actions/workflows/dotnet.yml/badge.svg)](https://github.com/iamfifya/AccuratSystem/actions/workflows/dotnet.yml)

## Описание проекта

**Accurat** — это клиент-серверный программный комплекс для автоматизации бизнес-процессов сети автомоек. Проект построен по принципу монорепозитория (monorepo) и включает в себя производительный десктопный рабочий терминал администратора и масштабируемый REST API сервер с централизованной базой данных.

Система позволяет управлять несколькими филиалами, вести сложный финансовый учет, контролировать кассовые смены, управлять расписанием боксов и автоматически рассчитывать заработную плату сотрудников с учетом авансов и минимальных ставок.

## Структура репозитория

Проект разделен на две основные директории:

*   **`AccuratPanelCarWashing/`** — Клиентское приложение. Десктопный интерфейс администратора, разработанный на WPF с применением паттерна MVVM и кастомных UI-компонентов.
*   **`Akkurat.WebAPI/`** — Серверная часть. REST API бэкенд на ASP.NET Core, инкапсулирующий всю бизнес-логику, вычисления и взаимодействие с базой данных PostgreSQL.

## Основные возможности

### Архитектура и Учет заказов
*   **Мультифилиальность:** Управление всей сетью моек из единой базы данных со строгим разделением прав и доступа по `BranchId`.
*   **Единый конвейер заказов:** Предварительные записи и фактические заказы реализованы как единая сущность в БД с динамической системой статусов. Это исключает дублирование данных и конфликты при конвертации записей.
*   **Синхронизация времени:** Сервер работает в эталонном UTC-времени, а клиентское приложение динамически конвертирует даты в локальные часовые пояса для корректного отображения.

### Финансовый модуль и Касса
*   **Контроль потоков:** Учет фактического остатка в кассе с разделением по типам оплат (Наличные, Карта, Перевод, QR-код).
*   **Транзакции:** Раздельный учет приходов, расходов на нужды мойки и инкассаций.
*   **Авансы и Зарплата:** Именная выдача авансов сотрудникам с автоматическим вычетом из итоговой зарплаты. Поддержка расчета ЗП как процента от заказа с доплатой до МРОТ (гарантия за выход).
*   **Чистая прибыль (ЧПКО):** Автоматический расчет реальной прибыли компании за вычетом всех расходов и выплат.

### Расписание и CRM
*   **Умное расписание:** API-проверка доступности моечных боксов с блокировкой пересечений времени при записи.
*   **Клиентская база:** Сохранение истории визитов, расчет LTV (Total Spent) и счетчик посещений. Глобальный поиск и автозаполнение данных при вводе государственного номера автомобиля.

### Отчетность и Экспорт
*   Генерация интервальных финансовых отчетов за любой период.
*   Экспорт аналитики, кассовых смен и зарплатных ведомостей в форматы Excel (через ClosedXML) и JSON.

## Технологический стек

**Backend (API):**
*   C# 13, ASP.NET Core (.NET 10)
*   PostgreSQL 12+
*   Entity Framework Core
*   Swagger / Scalar (документация API)

**Frontend (WPF):**
*   C# 7.3, WPF (.NET Framework 4.8)
*   ClosedXML (работа с Excel)
*   Newtonsoft.Json

## Требования к окружению и Запуск

### 1. Подготовка базы данных и сервера
1. Убедитесь, что установлен .NET 10 SDK и сервер PostgreSQL.
2. Перейдите в папку бэкенда:
   ```bash
   cd Akkurat.WebAPI
Откройте appsettings.json и настройте строку подключения DefaultConnection к вашей БД PostgreSQL.

Примените миграции Entity Framework для создания таблиц:

```Bash
    dotnet ef database update
```
Запустите сервер:

```Bash
    dotnet run
```
2. Запуск клиентского приложения
Откройте общее решение AccuratSystem.sln или AccuratSystem.slnx в Visual Studio.

Убедитесь, что в классе ApiService (в клиенте WPF) указан корректный базовый URL запущенного Web API сервера.

Соберите проект AccuratPanelCarWashing и запустите его.
