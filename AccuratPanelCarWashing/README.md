# Accurat - Панель управления автомойкой

[![.NET Build](https://github.com/iamfifya/AccuratPanelCarWashing/actions/workflows/dotnet.yml/badge.svg)](https://github.com/iamfifya/AccuratPanelCarWashing/actions/workflows/dotnet.yml)

## Описание проекта

**"Accurat"** — это клиент-серверный комплекс для управления сетью автомоек. Приложение переведено на современную распределенную архитектуру: производительный десктопный клиент (WPF) взаимодействует с надежным REST API бэкендом (ASP.NET Core), а данные надежно хранятся в PostgreSQL.

Система полностью автоматизирует бизнес-процессы: от записи клиента и контроля боксов до глубокого финансового учета, ведения кассы и сложного расчета заработной платы с учетом авансов и доплат. Проект поддерживает мультифилиальность, позволяя управлять несколькими точками с одного сервера.

## Основные функции

### Архитектура и Мультифилиальность (New!)
- **Сеть автомоек:** Централизованное управление несколькими филиалами из единой базы данных.
- **Единый конвейер заказов:** Предварительные записи и текущие заказы объединены в общую сущность с плавающей системой статусов, что исключает дублирование данных.
- **Синхронизация времени:** Строгий учет часовых поясов (эталонное время в UTC на сервере и локальное MSK-время в интерфейсе).

### Финансовый модуль и Касса
- **Контроль потоков:** Расчет фактического остатка в кассе в реальном времени с учетом всех типов оплат (Наличные, Карта, Перевод, QR-код).
- **Транзакции:** Раздельный учет приходов, расходов на химию и аренду, а также инкассаций по каждому филиалу.
- **Авансы:** Именная выдача авансов сотрудникам с автоматическим вычетом из итоговой заработной платы при закрытии смены.
- **Чистая прибыль (ЧПКО):** Автоматический расчет реальной прибыли компании за вычетом всех расходов и выплат мойщикам.

### Управление сменами и заказами
- Строгий контроль кассовых смен (система блокирует закрытие смены при наличии активных заказов).
- Управление 3 моечными боксами в реальном времени с API-проверкой на пересечение времени записей.
- Интеллектуальный расчет стоимости услуг в зависимости от выбранной категории кузова (1-4).
- Система наценок и скидок с обязательной фиксацией причины в чеке.

### Продвинутая отчетность
- **Интервальные отчеты:** Генерация сводной финансовой аналитики за любой выбранный период.
- **Зарплатная ведомость:** Прозрачный расчет ЗП (процент от заказа) с гарантированной доплатой сотруднику до МРОТ за выход на смену.
- Экспорт финансовых отчетов и аналитики в Excel (ClosedXML) и JSON-форматы для бухгалтерии.

### CRM и Клиентская база
- Автоматическое сохранение истории визитов, расчет LTV (Total Spent) и счетчик посещений.
- Глобальный поиск и автозаполнение данных (модель, номер, скидка) при повторном визите по гос. номеру автомобиля.

## Технологии и Стек

Проект разделен на два связанных репозитория: клиентский интерфейс и серверную инфраструктуру.

| Компонент | Технология |
|-----------|------------|
| **Клиент (UI)** | WPF (.NET Framework 4.8), C# 7.3, MVVM |
| **API Сервер** | ASP.NET Core Web API (.NET 10) |
| **База Данных** | PostgreSQL (Entity Framework Core) |
| **Сетевое взаимодействие**| `HttpClient`, REST API, `System.Net.Http.Json` |
| **Экспорт и Сериализация**| ClosedXML, Newtonsoft.Json |

## Системные требования

**Для рабочего места (WPF-клиент):**
- OS: Windows 7 / 8 / 10 / 11
- Runtime: .NET Framework 4.8 или выше

**Для сервера (Web API):**
- Runtime: .NET 10 SDK
- База данных: PostgreSQL 12+
